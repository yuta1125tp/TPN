# Installation

```shell
git clone https://github.com/decisionforce/TPN.git
```

## Requirements

- Linux
- Python 3.5+
- PyTorch 1.0+
- CUDA 9.0+
- NVCC 2+
- GCC 4.9+
- mmcv 0.2.10

## Install MMAction
(a) Install Cython
```shell
pip install cython
```
(b) Install mmaction
```shell
python setup.py develop
```
